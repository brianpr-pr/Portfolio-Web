# Portfolio de Brian Perez Perez

## Español

Este repositorio contiene mi portfolio profesional. Soy Brian Perez Perez, desarrollador web freelance en España, y creo aplicaciones y sitios web para negocios que quieren mejorar su presencia online, aumentar su visibilidad y atraer nuevos clientes.

Mi objetivo es desarrollar soluciones claras, modernas y fáciles de mantener. También creo sitios que permiten a sus propietarios gestionar el contenido sin tener que ocuparse directamente del alojamiento o del mantenimiento técnico.

### Sitio web

[Visitar el portfolio](https://brianpr-pr.github.io/Portfolio-Web/)

### Contenido y características

- Presentación de mi perfil y mis servicios como desarrollador freelance.
- Resumen de mis habilidades y tecnologías.
- Enlaces directos de contacto por correo electrónico y LinkedIn.
- Interfaz responsive para móvil, tablet y escritorio.
- Contenido disponible en español e inglés.
- Cambio de idioma integrado en la página.

### Tecnologías

- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- i18next
- Git y GitHub

### Ejecución local

```bash
git clone https://github.com/brianpr-pr/Portfolio-Web.git
cd Portfolio
npm install
```

Después, abre `index.html` en el navegador o sirve el directorio con un servidor web local.

## English

This repository contains my professional portfolio. I am Brian Perez Perez, a freelance web developer based in Spain, and I build web applications and websites for businesses that want to improve their online presence, increase visibility, and attract new clients.

My goal is to create clear, modern, and maintainable solutions. I also build websites that allow business owners to manage their content without dealing directly with hosting or technical maintenance.

### Website

[Visit the portfolio](https://brianpr-pr.github.io/Portfolio-Web/)

### Content and features

- Introduction to my profile and freelance development services.
- Overview of my skills and technologies.
- Direct contact links for email and LinkedIn.
- Responsive interface for mobile, tablet, and desktop.
- Content available in Spanish and English.
- Built-in language switcher.

### Technologies

- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- i18next
- Git and GitHub

### Local setup

```bash
git clone https://github.com/brianpr-pr/Portfolio-Web.git
cd Portfolio
npm install
```

Then open `index.html` in your browser or serve the directory with a local web server.
